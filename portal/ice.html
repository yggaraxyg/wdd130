About me
This big list is a showcase of all my In Class Exercises, or I.C.E. projects. Each have a small picture and a description. If you want to go to one, click on it's image. They're not in order of date, though.

Apostle Spotlight

This was one of the first projects I completed, just a simple picture gallery of the prophet and the apostles. Here, I first learned how to use grids.

Boarding pass

This is the very first ICE project I completed. I replicated the size and shape of a Delta Airlines boarding pass.

Bonus

There's a bit of a funny story behind this one. Amazon web services went down one day, and for a while nobody could open canvas. Our teacher brought up a random picture, and had us all do our best to replicate it.

Font Awesome

This project was fairly simple. this particular exercise was dedicated toward teaching us how to access simple symbols and use fonts.

Image Gallery

This image gallery assignment was simple: reduce the load time of the site. I spent a long time just reducing image quality in a photo editing program. it wasn't too difficult, just a bit repetitive. I learned how to speed up loadtimes in this project.

Grid Flags

This project was simple, but a bit difficult. all we were assigned to do was recreate the flags of several countries with Grids. I didn't 'learn' anything new but the practice increased and improved my use of Grid.

Halloween

This project was a bit of a break. We were given some HTML, and told to go wild with CSS. I made this silly graveyard-themed website.

Positioning

This project was a large exercise in Grid usage. I did learn how to put a block into text so it wraps around it, but for the most part I practiced grid use

Post

This post is the image I used to represent ICE projects! I did this one right after Font Awesome, as a way of learning how to use icons in a project. Completely aside from the project, I rather like the morbidly obese rat picture. it's unhealthy and I would never feed a rat that much, but I think it looks funny.

Prophet cards

This is another early project. from it, I learned the very basics of HTML and CSS.

Signup

This is the latest project I've completed. from it, I managed to glean some of the basics of forms.

And More...

Every single class we're assigned a new ICE project. Who knows what else we'll be assigned to make?
